## Project Features

### Current Capabilities
- Full-text indexing with Xapian for PAK content search
- RocksDB storage for icons and game objects
- Multi-tab editor interface with syntax highlighting
- Image viewer with Direct2D rendering (DDS, PNG, JPG, BMP, TGA, GIF, HDR)
- GR2 model viewer with Direct3D 11 rendering
- PAK explorer for browsing without extraction
- LSF viewer for Larian's LSF files
- Game Object and Icon explorers
- PAK Generator for mod packaging
